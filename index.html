Проекты

image-to-PDF

Описание: Конвертер изображений в PDF. Оно позволяет пользователям загружать изображениях и сохранять их в виде единого PDF-документа.

Технологии: Python, Tkinter, Pillow.

Функционал:

  • Добавление изображений.
  • Удаление выделенных изображений.
  • Сохранение выбранных изображений в виде единого PDF-документа.

expense-tracker

Описание: Трекер расходов на Django. Позволяет пользователям отслеживать свои расходы и управлять бюджетом.

Технологии: Python, Django, Django Rest Framework, PostgreSQL, Bootstrap, Docker, Docker-Compose.

Функционал:

  • Добавление, редактирование и удаление расходов.
  • Добавление, переименование и удаление категорий расходов.
  • Возможность устанавливать лимит расходов.
  • Формирование статистики: день, неделя, месяц, год.

Функционал API:

  • Добавление, изменение, удаление расходов, категорий расходов, профилей пользователей и получение информации о них.
  • Получение записей расходов и категорий расходов конкретных пользователей.

camera-parts-parser

Описание: Скрипт для парсинга каталога интернет-магазина с последующим экспортом данных в CSV и/или XLSX.

Технологии: Python, BeautifulSoup, Pandas, ua-generator.

Функционал:

  • Парсинг информации о товаре: название, стоимость, наличие.
  • Экспорт данных в формате CSV и/или XLSX.

content-telegram-bot

Описание: Телеграм-бот для составления содержания канала по тегам.

Технологии: Python, asyncio, Telethon, PostgreSQL, SQLAlchemy 2.0, asyncpg, python-dotenv.

Что уже реализовано:

  • Отправка приветственного сообщения пользователю.
  • Получение данных публичного канала по ссылке (id, username).
  • Запись канала в базу данных.
  • Парсинг постов канала.
  • Парсинг тегов в каждом из постов.
  • Запись постов и тегов в базу данных.
